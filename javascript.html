JavaScript (JS)

Origem: "Criado por Brendan Eich na Netscape em 1995 (originalmente chamado Mocha, depois LiveScript). Foi rapidamente renomeado para JavaScript (JS)."

Propósito Inicial: "Trazer interatividade e comportamento simples para o lado do cliente (o navegador), validando formulários e criando efeitos básicos sem precisar recarregar a página."

Evolução: "Após 2005 (com o Ajax), o JS se tornou o principal motor de sistemas web dinâmicos. As especificações ECMAScript (ES6/ES2015) introduziram Classes e Orientação a Objetos de forma robusta. Hoje, é usado tanto no Front-end (com React, Vue) quanto no Back-end (com Node.js)."

Função: "É o comportamento e a lógica. Interage com o DOM para manipular o HTML e CSS em tempo real, respondendo a eventos do usuário."
