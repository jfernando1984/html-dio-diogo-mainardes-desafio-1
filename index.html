O Início e os Pilares Fundamentais

A história da web começou de forma humilde, focada primariamente na distribuição de documentos. Tim Berners-Lee, no CERN, desenvolveu o HTML (HyperText Markup Language) e o HTTP (Hypertext Transfer Protocol) por volta de 1991. Naquela época, a web era puramente estática.

Um fato curioso é que, no início, era comum o uso da tag para estilizar, algo que hoje fazemos exclusivamente com CSS.

Um dos pilares do HTML é o elemento âncora, ou a tag , que permite a navegação entre páginas. Este é o conceito central que sustenta a World Wide Web.

A Tríade Essencial e a Dinâmica

Com o tempo, o design web evoluiu para a tríade:

  1. HTML A Estrutura Semântica (esqueleto).
  2. CSS: A Estilização e Design (aparência).
  3. JavaScript A Interatividade e Comportamento (ação).

A introdução do JavaScript em 1995 permitiu que as páginas fossem dinâmicas. Antes disso, a experiência de usuário era... digamos, mais rudimentar.

A Era da Orientação a Objetos

Nos últimos anos, a programação Front-end migrou para frameworks complexos e conceitos de Orientação a Objetos. O meu curso recente na DIO focou exatamente nisso.

A DOM é a representação em árvore dos elementos HTML na memória do navegador, permitindo a manipulação via JavaScript.

A arquitetura orientada a objetos é essencial para gerenciar a complexidade de sistemas modernos, como vimos ao escrever as classes de um jogo. — Referência do Desafio DIO
