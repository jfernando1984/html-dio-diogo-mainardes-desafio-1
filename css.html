CSS (Cascading Style Sheets)

Origem: Proposto pela primeira vez em 1994, mas o padrão foi formalizado em 1996 pelo W3C.

Propósito Inicial: Separar a estrutura (HTML) da apresentação visual (Estilo). Antes do CSS, todo o estilo era feito diretamente dentro das tags HTML (como a tag que edita fonte).

Evolução: O CSS2 padronizou o modelo de caixas (Box Model). O CSS3 (lançado a partir de 1999) introduziu módulos que permitiram layouts complexos (como Flexbox e Grid), responsividade (@media queries), sombras, e animações.

Função: É a aparência e o design. Controla cores, tipografia, espaçamento e, crucialmente, o layout (posicionamento dos elementos na tela).
